## 📌 서비스 한 줄 소개

- **SnapBook은 _예약과 문의를 채팅 기반으로 통합해주는 서비스_ 입니다.**
- 매장 사장님과 고객이 이미 익숙한 **채팅 UI** 안에서  
  예약 접수 · 시간 조율 · 디자인 상담 · 예약 내역 확인까지 한 번에 처리할 수 있습니다.

---

## 🎯 문제 인식

> “예약 날짜, 시간과 문의를 한 번에 볼 수는 없을까?”  
> “내가 원하는 디자인 사진은 채팅처럼 편하게 보낼 수 없을까?”

기존 예약 방식은
- 예약 폼은 따로,  
- 문의/사진 공유는 인스타 DM이나 카카오톡으로 따로 이뤄져  
**정보가 여러 곳에 흩어지고, 놓치는 예약이 생기기 쉽습니다.**

SnapBook은 이 과정을 **하나의 채팅방**으로 모아,
- 예약 내용과 문의 내역이 분리되지 않고
- 고객과 사장님 모두 **한 화면에서** 상황을 파악할 수 있게 합니다.

## 🧩 핵심 기능

1. **매장 예약 링크 접속**
   - 인스타그램 프로필 링크 **SnapBook 예약 링크**를 클릭해 진입합니다.

2. **채팅방 접속**
   - 매장별 전용 **채팅방**이 생성되어, 고객은 일반 메신저처럼 메시지를 남길 수 있습니다.

3. **채팅방에서 바로 예약**
   - 채팅방 안에서 바로 **예약 폼**을 작성하여 예약을 접수합니다.
   - 날짜/시간/옵션 선택과 함께,  
     원하는 스타일 사진을 **이미지 업로드**로 간편하게 전송할 수 있습니다.

4. **예약 확인 메시지**
   - 사장님이 예약을 **확정 / 거절 / 조정 제안**하면,
   - 그 결과가 **예약 상태 메시지 카드**로 채팅방에 표시되어  
     고객이 예약 현황을 한눈에 확인할 수 있습니다.

---

## 💡 SnapBook이 제공하는 가치

- ✅ **고객**  
  - 채팅·예약·사진 전송을 한 번에 해결  
  - 예약 내역을 채팅방에서 바로 확인
- ✅ **매장 사장님**
  - 예약/문의 기록이 한 채널로 모여 관리가 쉬움  
  - 예약 상태(접수·확정·취소)를 메시지 형태로 고객에게 명확하게 전달
